OpenCode AI Lab

A self‑hosted AI coding assistant that runs entirely on a single GCP VM. No API keys, no external inference service.

Quick start

git clone https://github.com/HiroyasuDev/GENAI.git
cd GENAI
chmod +x opencode_lab/vm_startup.sh
./opencode_lab/vm_startup.sh   # creates the VM startup script
./opencode_lab/setup_stack.sh  # installs dependencies & model
./opencode_lab/start_lab.sh    # launches tmux + llama‑server + OpenCode

The VM will be destroyed at 6 PM, so clone now!
